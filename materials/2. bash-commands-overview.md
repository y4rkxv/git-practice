### Основні Команди Терміналу (Bash/MinGW64)

### Вступ: Навіщо потрібен Git Bash?

**Git Bash** надає вам доступ до командного рядка (CLI), який є основним методом взаємодії з Git. Команди нижче є командами операційної системи (Bash), які необхідні для **навігації** та **управління файлами** у ваших проєктах.

### 1\. Навігація та Перегляд

| Команда          | Призначення                                                                                 | Ментальна модель                | Приклад                             |
| :--------------- | :------------------------------------------------------------------------------------------ | :------------------------------ | :---------------------------------- |
| **`ls`**         | **L**i**s**t. Переглядає файли та папки в поточній директорії.                              | Подивитися, що є в цій папці.   | `ls *.csv` (покаже лише файли .csv) |
| **`pwd`**        | **P**rint **W**orking **D**irectory. Показує повний шлях до папки, де ви зараз знаходитесь. | Де я зараз?                     | `pwd`                               |
| **`cd <папка>`** | **C**hange **D**irectory. Змінює поточну папку.                                             | Перейти в іншу папку.           | `cd /path/to/project`               |
| **`cd ..`**      | Перехід на один рівень директорії вгору.                                                    | Повернутися назад.              | `cd ..`                             |
| **`clear`**      | Очищує весь текст в терміналі для зручності.                                                | Почати роботу з чистого екрана. | `clear`                             |

### 2\. Створення та Управління Файлами

- **`touch <назва_файлу>`**
  - **Призначення:** Створює новий, порожній файл.
  - **Приклад:** `touch results.txt` (Створити файл для запису результатів)
- **`mkdir <назва_папки>`**
  - **M**a**k**e **Dir**ectory. Створює нову папку (директорію).
  - **Приклад:** `mkdir visualizations` (Створити папку для графіків)
- **`cp <джерело> <призначення>`**
  - **C**o**p**y. Копіює файл або папку.
  - **Приклад:** `cp data.csv data_backup.csv` (Копіювати набір даних)
- **`rm <файл>`**
  - **R**e**m**ove. Видаляє файл.
  - **Увага:** **`rm -r <папка>`** використовується для видалення папки разом із усім вмістом. Будьте обережні, видалення є остаточним.

### 3\. Пошук у Тексті

- **`grep <шаблон> <файл>`**
  - **Призначення:** Шукає конкретний текст (шаблон) усередині текстових файлів і виводить рядки, що містять збіг.
  - **Приклад:** `grep "ERROR" logs.txt` (Знайти всі рядки з помилками у логах)
