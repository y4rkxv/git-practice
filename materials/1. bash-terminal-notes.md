

##  Взаємодія з Операційною Системою через Командний Рядок

### 1. Суть Командного Рядка (CLI)

- **GUI vs CLI:**
  - **GUI (Graphical User Interface)**: Використання **миші** та іконок для виконання завдань (зручно для більшості користувачів).
  - **CLI (Command Line Interpreter)**: Використання **текстових команд** у терміналі (швидше та потужніше для розробників та системних завдань).
- **Призначення CLI:** Дозволяє **напряму** взаємодіяти з операційною системою, виконувати складні сценарії та **автоматизувати** задачі, які важко зробити через GUI.

---

### 2. Типи Командних Оболонок у Різних ОС

| Операційна Система       | Основна Оболонка (CLI)         | Потужніші Оболонки                                                    |
| :----------------------- | :----------------------------- | :-------------------------------------------------------------------- |
| **Windows**              | **Command Prompt** (`cmd.exe`) | **PowerShell** (сучасний, для складних скриптів та автоматизації)     |
| **Linux/macOS**          | **Bash** (Bourne Again Shell)  | **Zsh** (зазвичай)                                                    |
| **Windows для розробки** | **Git Bash** (імітує Bash)     | Використовується для роботи з Git та виконання Linux-подібних команд. |

#### 2.1. Огляд Основних Оболонок

- **Command Prompt (`cmd.exe`):** Базовий, стандартний інструмент Windows для основних операцій.
- **PowerShell:** Більш **сучасний** та потужний інструмент Windows. Дозволяє виконувати команди, недоступні в Command Prompt, та створювати складні скрипти автоматизації.
- **Bash (Bourne Again Shell):** **Стандартна** оболонка для більшості Unix-подібних систем (Linux, macOS). Вважається більш зручною та потужною для розробників.
- **Git Bash:** Інструмент, який встановлюється разом із Git у Windows. Це дозволяє користувачам Windows отримати доступ до **Bash-команд**, що є критичним для роботи з багатьма інструментами розробника.

---

### 3. Як відкрити CLI в різних ОС

| Операційна Система       | Спосіб                                 | Швидкі клавіші                                     |
| :----------------------- | :------------------------------------- | :------------------------------------------------- |
| **Windows (cmd.exe)**    | Меню "Пуск" або "Виконати"             | **`Win + R`**, потім `cmd`                         |
| **Windows (PowerShell)** | Меню "Пуск" або `Win + X`              | **`Win + X`**, потім "Windows PowerShell"          |
| **Linux**                | Додаток "Terminal"                     | **`Ctrl + Alt + T`** (для більшості дистрибутивів) |
| **macOS**                | Додаток "Terminal" через Spotlight     | **`Cmd + Space`**, потім ввести "Terminal"         |
| **Windows (Git Bash)**   | Натисніть правою кнопкою миші на папці | **"Git Bash Here"**                                |
